# Autolith 0.13.2

Autolith 0.13.2 tightens terminal behavior and expands syntax highlighting with
Clojure, Haskell, and OCaml support.

## Terminal experience

- Pressing Ctrl-C again while graceful cancellation is pending now restores the
  terminal and forces Autolith to exit with status 130. Durable conversations
  still print their exact resume command.
- Startup command tips now draw from fresh operating-system entropy instead of
  the random state captured in the preloaded image, so the same tip no longer
  appears on every launch.
- Raised the contrast of ordinary activity-row text while retaining the distinct
  model and effort colors.

## Syntax highlighting

- Updated ColorLisp to 0.2.0 with highlighting and pathname detection for
  Clojure `.clj`, `.cljc`, `.cljs`, and `.bb` files; Haskell `.hs` and
  `.hs-boot` files; OCaml `.ml` files; and OCaml interface `.mli` files.
- Added aliases and script shebang detection for the new languages.
- Updated Nix packaging to discover nested generated parsers, including OCaml's
  separate implementation and interface grammars.

## Compatibility notes

- No interactive command spellings or persisted data formats changed in this
  release.
